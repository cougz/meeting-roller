## How It Works

1. Click "Roll the Dice!" button
2. Watch the animated dice shake for 1.5 seconds
3. Get a random number from 1-20
4. The result indicates which presenter is next!

Perfect for team standups, retrospectives, or any meeting where you need a fair way to pick who goes next.
